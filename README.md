# Conocimiento deportivo (personal)

Repo para **guardar lo que vas aprendiendo** y **atarlo a lo que realmente hacés**, para que las ideas no queden sueltas y puedas ver sentido con el tiempo. No está pensado como bitácora ni log de entrenamientos.

## Cómo usarlo

1. **`conocimiento/`** — Notas por tema (un archivo por idea grande o por área). Cuando algo te cierre o quieras fijarlo, lo volcás acá en tus palabras.
2. **`contexto.md`** — Qué hacés hoy (actividades, ritmo, límites). Sirve de **mapa fijo** para relacionar: al leer una nota, podés preguntarte “¿esto aplica a X que yo hago?”.
3. **Relacionar sin log** — En cada nota podés usar una sección opcional, por ejemplo **“En mi caso”** o **“Con lo que hago”**, donde enlazás el concepto con una o más prácticas que ya describiste en `contexto.md`. Así el conocimiento queda anclado a tu vida, no a fechas de sesión.

## Estructura

```
conocimiento/     # ideas, conceptos, dudas resueltas, referencias
contexto.md       # qué hacés y bajo qué condiciones (para dar sentido a las notas)
```

## Convenciones sugeridas (todas opcionales)

- Un archivo por tema: `sueño-y-rendimiento.md`, `escalada-fuerza.md`, etc.
- Al final o al medio de una nota: bloque **“En mi caso”** con viñetas cortas (“esto me sirve cuando…”, “en bici / en el gym…”).
- Si dos notas se hablan entre sí, un enlace relativo: `[recuperación](./recuperacion.md)`.

---

Si una convención te frena, ignorala. El objetivo es que acumular conocimiento te organice la cabeza, no que mantengas un diario.
