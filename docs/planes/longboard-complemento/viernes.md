# Viernes — Circuito A o B + Fuerza

🛹 Longboard como siempre · Referencia completa: [plan](./plan-complemento-casa-longboard.md)

> Elegí el circuito que más te pida el cuerpo. El bloque de fuerza va siempre al final.
