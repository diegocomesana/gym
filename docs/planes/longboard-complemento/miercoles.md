# Miércoles — Circuito A + Fuerza 🔝

🛹 Longboard como siempre · Referencia completa: [plan](./plan-complemento-casa-longboard.md)

> **Pico de la semana.** Es el día de mayor carga. Llegás con dos días de trabajo encima pero sin el valle del jueves todavía — el sistema nervioso está cargado pero no roto. Acá va el estímulo más exigente.
